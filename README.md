## 6. DIRECCIÓN DE DISEÑO WEB

### Estética

**Industrial-Refinada Oscura** — Dark luxury con toques de rojo de precisión.

Inspiración: estudios de arquitectura europeos, agencias de diseño premium latinoamericanas.

### Elementos de diseño

- Fondo oscuro dominante (`#0D0D0D`)
- Líneas finas blancas como elementos decorativos (guiño al nombre "Línea Fina")
- Bloques de color rojo para separadores y CTAs
- Grids asimétricos para las secciones de servicios
- Tipografía condensada bold para títulos impactantes
- Micro-animaciones en scroll (reveal de elementos)
- Cursor personalizado (línea fina — opcional)
- Efecto de "líneas de plano arquitectónico" como textura de fondo (sutil)

### Responsive

- Mobile-first (la mayoría de clientes llegarán por móvil)
- Nav hamburger en mobile
- Cards apiladas en mobile
